# Kotae CS - 한국 기업을 위한 일본 시장 고객 지원 대행 서비스

## 📌 프로젝트 개요

**Kotae CS(코타에 CS)** 는 한국 기업이 일본 시장에 진출할 때 필요한 고객 지원 업무를 대행하는 서비스의 웹사이트입니다.

### 태그라인
**"모든 질문에, 답을 드립니다."** / *"Every question, answered."*

---

## 🎯 서비스 컨셉

### 타겟 고객
- 일본 시장에 진출하는 한국 기업 (K-beauty, K-POP, EC 등)
- 일본어 고객 지원에 어려움을 겪는 한국 스타트업~중견 기업

### 차별화 포인트
1. **한일 바이링구얼 완벽 대응** - 일본 고객 대응 + 한국 본사 보고
2. **K-beauty/K-POP 업계 전문 지식** - 화장품 성분, 약사법, 아이돌 굿즈 대응
3. **Zendesk 표준 장착** - 확장 가능한 운영 기반
4. **스몰 스타트 가능** - 월 80만원부터 시작할 수 있는 유연한 플랜

---

## 📁 사이트 구성

### 페이지 목록
1. **index.html** - 홈페이지 (히어로 섹션 + 서비스 개요)
2. **service.html** - 서비스 내용 페이지 (상세한 대응 내용)
3. **pricing.html** - 요금 페이지 (5개 플랜 + 비교표)
4. **about.html** - 회사 소개 페이지 (미션·비전)
5. **contact.html** - 문의 페이지 (무료 상담 폼)

### 디렉토리 구조
```
.
├── index.html              # 홈페이지
├── service.html            # 서비스 내용
├── pricing.html            # 요금 플랜
├── about.html              # 회사 소개
├── contact.html            # 문의
├── css/
│   ├── style.css          # 메인 스타일
│   └── pricing.css        # 요금·서비스 페이지 추가 스타일
├── js/
│   └── main.js            # JavaScript (햄버거 메뉴, 폼 등)
├── test-form.html         # Formspree 테스트용 폼
├── FORMSPREE_SETUP.md     # Formspree 설정 가이드
└── README.md              # 이 파일
```

- 대응 시간: 평일 9:00-19:00
- 월간 개선 미팅
- **첫 달 50% OFF**

### Premium 플랜 (월액 550만원)
- **타겟**: 일본 시장에서 본격 전개하는 한국 기업
- 월 600건까지 (이메일·채팅·SNS 합산, 유연 배분 가능)
- 전담 담당자 배정
- 대응 시간: 평일 9:00-21:00 + 토요일 대응
- 약사법 대응 지원
- 상급자 대응

### Premium Plus 플랜 (월액 800만원)
- **타겟**: 대규모 전개 기업
- 월 1,200건까지

### Enterprise 플랜 (견적 문의)
- **타겟**: 월 1,500건 이상

- 완전 커스터마이징

### 특별 할인
